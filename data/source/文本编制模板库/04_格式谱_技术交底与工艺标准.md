# 04 格式谱 — 技术交底 / 工艺标准

学自：`技术交底大全[500份＋]\01土方工程\` 等共 1384 份工艺标准。

适用：分项工程技术交底、班前作业指导书、操作规程、首件工艺标准。

---

## 一、典型章节框架（紧凑型 7 节）

```
封面（短，1 段即可）
  XX 工程技术交底卡 / XX 工艺标准
  交底日期：____年__月__日  交底人：_____

1. 适用范围
2. 施工准备
   2.1 材料要求
   2.2 主要机具
   2.3 作业条件
3. 操作工艺
   3.1 工艺流程
   3.2 操作要点
4. 质量标准
   4.1 主控项目
   4.2 一般项目
   4.3 允许偏差
5. 成品保护
6. 安全与环境
   6.1 安全注意事项
   6.2 环境注意事项
   6.3 文明施工
7. 交底记录
   - 交底人签字
   - 接受交底人名单（含工种/工龄）
```

> 学习观察：1384 份样本均按"工艺标准（101-1996）"等带编号的统一格式（住建部 JGJ/T 3040-2013 体系），编号"101/102/103…"对应分项工程序号。

---

## 二、段落编号规范

- 节：1 / 2 / 3 / 4
- 子节：2.1 / 2.2
- 子子节：2.1.1
- 操作要点：1）2）3）或 ① ② ③
- 质量：主控项目用"主控项："前缀

---

## 三、必含字段与表述模板

### 适用范围（标准开场）

```
本工艺标准适用于 ____________________ 工程 ____________________ 分项
（如：本标准适用于工业与民用建筑的人工挖土工程）
```

### 施工准备 — 3 类资源

**材料要求：**

| 材料 | 规格型号 | 进场要求 | 检验方法 |
|---|---|---|---|
| | | | |

**主要机具：**

| 机具 | 规格 | 单位 | 数量 | 用途 |
|---|---|---|---|---|
| | | | | |

**作业条件：**

1. ____________________
2. ____________________
3. ____________________

### 操作工艺 — 必写工艺流程图

```
1. 工艺流程
   准备 → A → B → C → 检验 → D → 验收

2. 操作要点
   1）操作 A
   2）操作 B
   3）操作 C
```

### 质量标准 — 严格分主控/一般

```
4.1 主控项目（强制性）

   1）__________必须符合__________要求。（检验方法：____；检查数量：____）
   2）__________不允许__________。

4.2 一般项目

   1）__________应基本__________。
   2）允许偏差如表：
   
   | 项目 | 允许偏差 | 检验方法 |
   |---|---|---|
```

### 安全与环境（必含 12 项核心）

```
6.1 安全注意事项
   1) 进入现场戴安全帽，高空作业系安全带
   2) 安全员持证上岗
   3) ____________________
   4) ____________________
   5) ____________________

6.2 环境注意事项
   1) 控制扬尘（洒水、覆盖）
   2) 控制噪声（夜间不超 55 dB）
   3) 废弃物分类堆放
   4) ____________________

6.3 文明施工
   1) 现场整洁、工完场清
   2) ____________________
```

## 四、写作自检清单

- [ ] 标题含分项工程名 + "工艺标准"
- [ ] "适用范围"明确分项工程与单位工程
- [ ] "施工准备"列 3 类（材料/机具/作业条件）
- [ ] "工艺流程"用箭头单线图
- [ ] "操作要点"按 1）2）3）分点
- [ ] "质量标准"主控/一般分两条
- [ ] "允许偏差"用表格
- [ ] "安全与环境"含至少 5 项安全 + 4 项环境
- [ ] 底部有交底人签字 + 接受交底人名单
- [ ] 引用规范版次准确
