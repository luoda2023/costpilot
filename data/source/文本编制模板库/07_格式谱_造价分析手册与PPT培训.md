- 案例编号：案例 1 / 案例 2 或 1-1 / 2-3

## 四、写作自检清单

### 定额分析手册

- [ ] 目录按"章-子目"两层级展开
- [ ] 每个子目必含四部分（定额内容/单价组成/含量分析/注意事项）
- [ ] 综合单价与 `价格信息库` 同口径
- [ ] 注意事项配示意图（必要时）
- [ ] 编号与官方定额编号一致（如 1-57）

### 案例汇编

- [ ] 每个案例五要素（概况/起因/理由/过程/结果）+ 经验教训
- [ ] 处理过程含时间链（意向/报告/核查/审批）
- [ ] 处理结果分"索赔金额"和"核准金额"对比
- [ ] 案例间引用证据时间/编号准确

### 错漏防范手册

- [ ] 分部分类清晰（主体/装修/空调/智能化/电/消防/园建/泛光/其他）
- [ ] 每条"审计意见 + 解析"成对出现
- [ ] 解析四段（起因/规范/方法/实例）齐全
- [ ] 实例数据来自真实项目

### PPT 课件

- [ ] 封面含培训主题/主讲人/日期
- [ ] 第一页目录
- [ ] 每页只讲 1 个要点（≤6 行字）
- [ ] 关键数据/图表配图
- [ ] 案例穿插于讲解段中
- [ ] 尾页含总结 + 互动
