- `14\17\69\` 装配式建筑估算指标

## 一、典型章节框架

### A. 投资估算（可研阶段）

```
封面：XXXX 工程投资估算
  编制单位 / 编制人 / 审核人 / 审定人 / 日期

第一部分 编制说明
  1.1 项目概况
  1.2 编制依据
    - 项目建议书批复
    - 估算指标（如《城市综合管廊工程投资估算指标》）
    - 当地信息价（年月份）
    - 类似工程经济指标
  1.3 编制方法（指标估算法 / 类似工程比较法 / 概算指标法）
  1.4 主要假设条件
  1.5 估算结果汇总

第二部分 投资估算汇总表
  2.1 工程费用
    - 建筑工程费
    - 安装工程费
    - 设备购置费
  2.2 工程建设其它费
    - 建设单位管理费
    - 可研/设计/监理费
    - 勘察费
    - 招标代理费
    - 环境评价费
    - 节能评估费
    - 社会稳定风险评估费
    - 工程保险费
    - 联合试运转费
    - 安全生产措施费
    - 文明施工费
  2.3 预备费
    - 基本预备费（5~8%）
    - 价差预备费
  2.4 建设期利息
  2.5 铺底流动资金
  2.6 项目总投资

第三部分 单位工程估算明细
  3.1 土建工程估算明细
  3.2 安装工程估算明细
  3.3 设备购置明细

第四部分 投资分析
  4.1 单位投资分析（元/㎡、元/m·d、元/t 等）
  4.2 与类似工程比较分析
  4.3 主要技术经济指标

第五部分 资金筹措与使用计划
  5.1 资金来源（自筹/贷款/财政）
  5.2 年度资金使用计划
```

### B. 目标成本测算（房企产品级）

```
封面：XXXX 项目目标成本测算
  - 项目名称 / 项目编号 / 编制部门 / 日期

第一部分 项目基础信息
  1.1 项目区位
  1.2 占地面积
  1.3 容积率 / 总建筑面积 / 计容面积
  1.4 产品类型（高层 / 多层 / 别墅 / 商业 / 车库）
  1.5 销售价预估

第二部分 成本测算汇总
  2.1 土地成本（含契税/印花税）
  2.2 前期工程费（含可研/设计/勘察/报建）
  2.3 建安工程费
    - 主体土建
    - 装饰装修
    - 门窗栏杆
    - 水电安装
    - 消防
    - 智能化
    - 电梯
    - 室外配套
  2.4 配套设施费
  2.5 工程建设其它费
  2.6 预备费
  2.7 财务费用（建设期利息）
  2.8 销售费用
  2.9 管理费用
  2.10 税金（增值税/土地增值税/所得税）
  2.11 目标利润
  
  总成本合计 / 总销售收入 / 测算利润率

第三部分 建安成本明细
  3.1 各产品类型分别测算
  3.2 单方造价（元/㎡）分专业汇总
  3.3 限额设计指标（钢筋 kg/㎡、砼 m³/㎡）

第四部分 成本指标分析
  4.1 与典型项目对标（参考 `价格信息库\` 中各专业速查表）
  4.2 与历史项目对标（参考 `13\` 历史合同价）
  4.3 优化建议
```

### C. 设计概算

```
封面：XXXX 工程设计概算
  编制单位 / 编制人 / 复核人 / 审定人 / 日期

第一部分 编制说明
  1.1 项目概况
  1.2 编制依据
    - 初步设计图纸
    - 概算定额 / 概算指标
    - 当地信息价
    - 估算批复文件
  1.3 主要技术经济指标

第二部分 总概算
  2.1 工程费用
  2.2 其它费用
  2.3 预备费
  2.4 建设期利息
  2.5 总投资

第三部分 综合概算
  3.1 各单位工程概算汇总

第四部分 单位工程概算
  4.1 建筑工程概算
  4.2 安装工程概算
  4.3 设备购置概算

第五部分 主要材料用量
```

## 二、段落编号规范

- 部/章/节：第一部分 → 第一章 → 1.1 → 1.1.1
- 表号：表 1-1（部分-序号）
- 金额单位：万元（保留 2 位小数）/ 元
- 单方指标：元/㎡ / 元/m³ / 元/t·d

## 四、写作自检清单

### 投资估算自检

- [ ] 编制依据含可研批复 + 估算指标 + 信息价月份
- [ ] 工程费用按"建筑工程/安装工程/设备购置"分列
- [ ] 工程建设其它费 11 项齐全（含社会稳定风险评估）
- [ ] 预备费率取 5~8%
- [ ] 建设期利息按实际贷款利率计算
- [ ] 铺底流动资金按 30% 流动资金计
- [ ] 资金来源与使用计划表
- [ ] 与类似工程经济指标对标

### 目标成本测算自检

- [ ] 项目基础信息含容积率/计容面积
- [ ] 11 大成本科目齐全
- [ ] 建安成本按"分专业 × 单方造价"展开
- [ ] 限额设计指标附后
- [ ] 与典型项目对标（引用 `价格信息库\` 中具名文件）
- [ ] 测算利润率与销售单价对比

### 设计概算自检

- [ ] 三级概算结构（总/综合/单位）齐全
- [ ] 概算定额/指标配年份
- [ ] 单位工程概算与综合概算数字勾稽
- [ ] 主要材料用量表
- [ ] 主要技术经济指标
- [ ] 编制/复核/审定三级签名
