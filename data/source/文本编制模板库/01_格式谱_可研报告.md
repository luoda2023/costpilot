## 二、段落编号规范

- 章：第一章 / 第二章…（中文数字）
- 节：1.1 / 1.2 / 1.3
- 子节：1.1.1 / 1.1.2
- 子子节：1.1.1.1
- 表/图编号：表 5-1（第 5 章表 1）/ 图 5-1
- 单位：统一中文（m³/d，t/d，ha，km，㎡）；金额统一万元，保留 2 位小数

## 四、写作自检清单

- [ ] 封面含项目名 + 报告类型 + 编制单位 + 日期
- [ ] 扉页含证书等级 + 院长/总工/项目负责人
- [ ] 11~14 章齐全（小项目可合章但需保留"总论/建设条件/必要性/方案/环境/节能/实施/估算/经济/结论/招标"骨架）
- [ ] "编制依据"含 4 类（法规/规章/规范/项目文件）
- [ ] "主要经济指标"必含 8 项（规模/总投资/单位投资/年运行成本/单位运营成本）
- [ ] "方案比选"必有 3 方案对比表，单价引用 `价格信息库` 实价
- [ ] "投资估算"必含一二三类费用 + 预备费 + 利息 + 流动资金，并附"资金筹措表"
- [ ] "招标方案"含招标范围/组织/方式（依据 2018 国家发改委 18 号令）
- [ ] "社会稳定风险分析"必含（建标[2013]123 号文要求）
- [ ] 单位口径与工程量表一致（必查 `查询存档/05_UNIT_AUDIT.md`）
- [ ] 投资估算单价引到 `价格信息库` 中具名文件
