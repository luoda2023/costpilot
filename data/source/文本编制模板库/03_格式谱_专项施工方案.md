# 03 格式谱 — 专项施工方案

学自：
- 蓄水池施工方案.docx
- 冬期施工组织设计.docx
- 冲孔灌注桩施工方案.docx
- 塔吊专项施工方案.docx

适用：分部分项工程专项方案、专家论证方案、季节性方案、应急预案。

---

## 一、典型章节框架（15~18 节）

```
封面
  方案名称（XXXX 工程 XXXX 专项施工方案）
  编制人 / 审核人 / 审批人
  编制日期

第 1 节 工程概况
  1.1 项目概况
  1.2 方案涉及范围
  1.3 相关方责任主体
  1.4 地质条件（含勘察资料编号）
  1.5 周边环境

第 2 节 编制依据
  2.1 设计图纸（编号/版次）
  2.2 施工规范（含强制性条文）
  2.3 验收标准
  2.4 公司管理体系文件
  2.5 类似工程经验

第 3 节 安装位置平面和立面图（按需）
第 4 节 所选用的设备/材料型号及技术参数
第 5 节 基础和附着装置设置（按需）
第 6 节 爬升工况及附着节点详图（按需）
第 7 节 安装顺序和质量要求
第 8 节 主要部件重量和吊点位置
第 9 节 安装辅助设备型号、性能及布置
第 10 节 电源设置
第 11 节 施工人员配置
第 12 节 吊索具和专用工具配备
第 13 节 安装工艺程序
第 14 节 施工工艺及技术措施
第 15 节 安全装置测试 / 质量控制要点
第 16 节 重大危险源和安全技术措施
第 17 节 应急预案
第 18 节 附件（计算书、资格证书、出厂资料）
```

> 注：以上为完整结构。简单方案可合并相邻小节（如塔吊方案用全 18 节，混凝土方案只需 8 节）。

---

## 二、段落编号规范

- 节：第一节 / 第二节（节序）
- 子段：1. / 2. / 3.
- 子子段：1） 2） 3） 或 (1) (2) (3)
- 表/图：表 1-1 / 图 1-1（节序-序）

---

## 三、必含字段与表述模板

### 工程概况 — 必含 5 字段

```
项目名称：__________________________________
工程建设地点：________________________________
周边环境：__________________________________
总建筑面积：__________ ㎡
占地面积：__________ ㎡
本方案涉及范围：________________________________
```

### 编制依据 — 必引 3 类

```
1. 设计文件
   - 《XXX 工程施工图纸》（编号：__________，版次：______）
2. 施工规范（按专业选）
   - 《建筑工程施工质量验收统一标准》GB50300-2013
   - 《混凝土结构工程施工规范》GB50666-2011
   - 《混凝土结构工程施工质量验收规范》GB50204-2015
3. 地勘资料、专家评审意见
```

### 施工工艺流程 — 必写流程图

```
场地平整 → 放线定桩位 → 桩机就位 → 埋护筒
       → 钢筋笼制作 → 冲进 → 掏渣
       → 冲进掏渣循环至设计深度
       → 吊放钢筋笼 → 清渣 → 浇灌桩身混凝土
```

### 质量控制要点 — 必有表

| 控制项 | 允许偏差/要求 | 检验方法 | 检查频次 | 责任人 |
|---|---|---|---|---|
| 桩位 | ±10mm | 全站仪 | 全数 | 测量员 |
| 桩径 | ≥设计 | 钻孔测 | 全数 | 班组长 |
| 桩长 | ≥设计 | 钻孔测 | 全数 | 监理旁站 |
| 混凝土强度 | 达设计等级 | 留试块 | 每 50m³ 一组 | 试验员 |

### 应急预案 — 必含 6 类

```
应急组织架构：组长（项目经理）/副组长/组员
应急物资：__________（清单附后）
应急联系电话：__________
应急响应程序：发现→汇报→救援→调查→处理→恢复
应急培训/演练：每季度 1 次
重大风险点提示：__________
```

### 计算书（专项方案必备）四类典型

1. **模板/支撑计算**：荷载 + 立杆稳定性 + 跨中挠度
2. **脚手架计算**：立杆稳定 + 连墙件 + 风荷载
3. **塔吊基础计算**：基底压应力 + 抗倾覆 + 抗滑移
4. **深基坑支护计算**：土压力 + 支护桩/锚杆 + 整体稳定

## 四、写作自检清单

- [ ] 工程概况含项目基本信息 + 方案涉及范围 + 周边环境
- [ ] 编制依据含规范的年号版次
- [ ] 工艺流程图清晰（箭头单线表达）
- [ ] 操作要点按"1）/2）/3）"分点写
- [ ] 质量控制点必有"控制项/允许偏差/检验方法/检查频次/责任人"
- [ ] 重大危险源清单 + 控制措施
- [ ] 应急组织含项目经理为组长 + 应急联系电话
- [ ] 计算书必有（涉及安全的方案）+ 计算引用规范
- [ ] 附件含资格证书（起重工、电工、焊工等）
- [ ] 引用规范版次准确（GB50300-2013 而非 -2001）
