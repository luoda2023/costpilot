## 一、代写服务标准流程（你给我任务，我按这个走）

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 定类型 ─── 你一句话告诉我"要写什么"（可研/方案/合同/交底/结算书/概算…）  │
│   ↓                                                          │
│ 2. 选模板 ─── 我去 文本编制模板库\ 检索对应"格式谱"                      │
│   ↓                                                          │
│ 3. 问要素 ─── 我用 AskUserQuestion 一次性问你 4 项必填要素：            │
│              (1) 项目名+地点                                  │
│              (2) 工程规模（面积/长度/管径/层数等）                  │
│              (3) 编制阶段（可研/初设/施工图/结算）                  │
│              (4) 输出格式（docx/md/纯文本） + 是否要含计算表       │
│   ↓                                                          │
│ 4. 查数据 ─── 我从 价格信息库\ 取单价，从 信息价_分省市 取当地信息价，  │
│              从 规费措施税金 取费率，按 06_COST_COMPOSITION 算费用        │
│   ↓                                                          │
│ 5. 出稿 ──── 按该类型"格式谱"逐章节填写，保留行业段落习惯             │
│              （章节序、编号制、术语口径与原文样本一致）                │
│   ↓                                                          │
│ 6. 自检 ──── 对照该类型的"必检项"（每类末尾列出）逐条核对               │
│   ↓                                                          │
│ 7. 交付 ──── 直接给出 .md / 写入 .docx；必要时附"数据来源追溯表"       │
└─────────────────────────────────────────────────────────────┘
```

### 关键原则

1. **不空写**：所有数据有出处，每条单价引用 `价格信息库` 中的具名文件
2. **不乱排版**：章节序号严格对齐行业惯例（参见 `01_格式谱_可研报告.md` ~ `08_格式谱_目标成本与概算.md`）
3. **不省略必检项**：交底必含"质量标准/安全事项"，施组必含"施工部署/进度计划"，概算必含"二类三类费用"
4. **可改稿**：交付后你说要扩要简，我都保留章节骨架

每份"格式谱"包含 4 个固定小节：
1. **典型章节框架**（实地学到的目录树）

3. **必含字段与表述模板**（填空即用）
4. **写作自检清单**（交付前的勾选检查）

**典型一次代写的取用路径**：

> 你说"给我写一份某 DN800 污水管网改造的可研报告"
> 1. 我用 `01_格式谱_可研报告.md` 拿到 11 章框架
> 2. 用 `查询存档\07_PIPELINE_EXCAVATION_PRICE` 拿到 DN800 管道单价区间
> 3. 用 `价格信息库\市政\重点专题_管道基坑钢板桩.md` 拿到钢板桩/降水单价
> 4. 用 `价格信息库\信息价_分省市\省市信息价主线表.md` 拿到所在地信息价
> 5. 用 `价格信息库\规费措施税金\规费_措施费_税金费率表.md` 拿到当地规费率
> 6. 用 `查询存档\06_COST_COMPOSITION.md` 算出估算总造价
> 7. 按 `08_格式谱_目标成本与概算` 第 4 章把估算表插入可研第十一章"投资估算与资金筹措"
> 8. 自检可研"必检项"全勾 → 出稿
