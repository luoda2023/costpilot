## 一、典型章节框架（合同示范文本结构）

```
封面 / 标题
  XXXX 合同（或合同编号）

第一部分 合同协议书
  发包人(全称)：_______________________________
  承包人(全称)：_______________________________
  鉴于 ____________________ 依照《中华人民共和国合同法》《中华人民共和国建筑法》《中华人民共和国招标投标法》…
  一、工程概况
  二、工程承包范围
  三、合同工期（开工/竣工/总日历天数）
  四、质量标准
  五、合同价款（金额形式/暂列金额/暂估价）
  六、组成合同的文件（合同协议书/中标通知书/投标函/专用条款/通用条款/图纸/工程量清单/标准规范）
  七、词语定义
  八、承诺（不违规转包/不拖欠工资/保密）
  九、签订时间
  十、签订地点
  十一、补充条款
  十二、合同生效
  发包人（公章）      承包人（公章）
  法定代表人/委托代理人签字

第二部分 通用合同条款
  1. 一般约定（词语定义/通知/保密/知识产权）
  2. 发包人义务
  3. 承包人义务
  4. 监理人
  5. 工期
  6. 质量与验收
  7. 合同价款与支付
  8. 变更与调整
  9. 不可抗力
  10. 知识产权
  11. 违约责任
  12. 索赔与争议解决
  13. 合同生效与终止
  14. 合同解除

第三部分 专用合同条款
  - 各地/各项目细化、补充、修改通用条款
  - 编号与通用条款对齐
  - 横道 ___ 处可填具体项目要求；如无约定填"无"或"/"

附件
  - 附件 A：中标通知书
  - 附件 B：工程量清单
  - 附件 C：技术要求 / 设计文件
  - 附件 D：图纸清单
  - 附件 E：履约担保格式
  - 附件 F：支付担保格式
  - 附件 G：廉政协议
```

## 二、段落编号规范

**两套并存：**

- **示范文本结构**：第一部分 / 第二部分 / 第三部分 → 编号 1./2./3. → 1.1/1.2 → (a)(b)(c)
- **简单采购合同**：一、二、三、四（按条款列，无章节嵌套）

## 三、按合同类型细化

### A. 采购合同（购销类，简单）

```
XXXX 购销合同

供方：__________________
法定代表人：___________
住所地：________________
联系电话：______________

需方：__________________
法定代表人：___________
住所地：________________
联系电话：______________

兹有需方与供方就 ________ 工程施工需要订购下列产品事宜，
经友好协商达成如下购销协议，特此签订合同。

第一条   标的（供应规格、数量、价格、金额）
  注：以上材料供方提供发票为：税率为 13% 增值税专用发票；需方见票付款。

第二条   质量要求和验收标准
第三条   交货时间地点及费用
第四条   包装（标准、费用）
第五条   检验与异议处理
第六条   结算方式
第七条   违约责任
第八条   不可抗力
第九条   争议解决
第十条   合同生效与份数

供方（公章）                  需方（公章）
法定代表人：________        法定代表人：________
日期：____年__月__日         日期：____年__月__日
```

### B. 劳务分包合同模板

```
建设工程施工劳务分包合同

工程承包人（施工总承包人或专业工程承（分）包人）：__________
劳务分包人：__________

依照《中华人民共和国合同法》《中华人民共和国建筑法》及其它有关法律、行政法规，遵循平等、自愿、公平和诚实信用的原则，鉴于 ____ （发包人）与承包人签订合同协议书，双方就劳务分包事项协商一致，订立本合同。

1、劳务分包人资质情况
   资质证书号码：__________
   发证机关：__________
   资质专业及等级：__________
   复审时间及有效期：__________

2、劳务分包工作对象及提供劳务内容
   工程名称：__________
   工程地点：__________
   分包范围：__________
   提供分包劳务内容：__________

3、分包工作期限
   开始工作日期：____年__月__日
   结束工作日期：____年__月__日
   总日历工作天数：____ 天

4、质量标准
   工程质量：按总（分）包合同有关质量的约定、国家现行的《建筑安装工程施工及验收规范》和《建筑安装工程质量评定标准》，本工作必达到质量评定 ____ 等级。

5、合同文件及解释顺序
   （1）本合同
   （2）本合同附件
   （3）本工程施工总承包合同
   （4）本工程施工专业承（分）包合同

6、标准规范
7、劳务报酬
8、工时制与节假日
9、劳务工作量的核对与确认
10、劳务报酬的支付
11、施工机具与材料
12、质量检查与验收
13、安全施工与事故处理
14、违约责任
15、争议解决
16、合同生效与终止
```

## 四、写作自检清单

- [ ] 标题含合同类型 + 项目名
- [ ] 双方主体完整（名称 + 法定代表人 + 住所地 + 联系电话 + 公章位）
- [ ] 鉴于引言：依据相关法律（合同法/建筑法/招投标法）
- [ ] 五要素齐全：工程范围/工期/质量/合同价款/文件构成
- [ ] 通用条款与专用条款编号一致
- [ ] 附件清单完整（中标通知/清单/图纸/担保）
- [ ] 签订时间/地点/份数明确
- [ ] 公章位与法定代表人签字位齐全
- [ ] 引用法律名准确（《民法典》替代了《合同法》，2021.1.1 起）
- [ ] 涉及价格的条款与 `价格信息库` 单价/计价方式相容
