## 一、典型章节框架

### A. 工程结算书（项目级）

```
封面：XXXX 工程结算书
  建设单位 / 施工单位 / 编制日期 / 编制人 / 复核人 / 审定人

第一部分 工程概况
  1.1 项目概况（结构/规模/工期）
  1.2 结算依据（合同编号/变更/签证清单/竣工图）
  1.3 主要结算指标

第二部分 结算编制说明
  2.1 编制范围
  2.2 编制依据
  2.3 主要材料价格取定（来源信息价/合同价）
  2.4 人工费、机械费、规费取定
  2.5 调整事项说明（变更/签证/索赔/进度奖）
  2.6 争议事项及处理建议

第三部分 结算汇总表
  3.1 工程造价结算汇总表
  3.2 单位工程结算汇总表
  3.3 措施项目费结算表
  3.4 其他项目费结算表
  3.5 规费/税金结算表

第四部分 结算工程量及单价明细
  4.1 分部分项工程量清单计价表
  4.2 工程量计算书
  4.3 综合单价分析表
  4.4 主要材料价格取定表

第五部分 设计变更、签证、索赔清单
  5.1 设计变更汇总表
  5.2 现场签证汇总表
  5.3 索赔与反索赔清单
  5.4 专项争议处理记录

附件
  - 竣工图纸
  - 验收记录
  - 主要材料合格证
  - 现场签证原件
  - 设计变更单
  - 索赔意向通知 / 索赔报告
```

## 四、写作自检清单

### 结算书自检

- [ ] 工程概况含规模/工期/质量评定结果
- [ ] 编制依据含合同号 + 竣工图号 + 主要签证号
- [ ] 调整事项逐项编号 + 金额
- [ ] 主要材料价格取定附信息价来源月份
- [ ] 汇总表与明细表数字勾稽正确（横向/纵向）
- [ ] 综合单价分析表中规费、税金算法正确
- [ ] 附件齐全（竣工图、签证、变更）
- [ ] 编制/复核/审定三级签字

### 审计方案自检

- [ ] 范围与对象明确
- [ ] 审计依据含法规 + 合同 + 资料
- [ ] 审计内容覆盖 7 类（量/价/措施/变更/签证/索赔/材料价差）
- [ ] 审计方法明确（全面/重点/抽样）
- [ ] 时间安排含阶段节点
- [ ] 审计风险与应对措施

### 索赔报告自检

- [ ] 事件概述五要素（名称/时间/部位/责任方/后果）
- [ ] 索赔理由三依据（合同/法律/事实）
- [ ] 索赔金额计算清单完整（人工/材料/机械/管理/利润/税金）
- [ ] 工期索赔单独章节
- [ ] 附件含意向通知 + 签证 + 照片 + 考勤
- [ ] 引用合同条款精确到"第X条第X款"
