## 二、查价工作流（用户给工程量表后的标准流程）

### 第 1 步：识别专业

清单内有"管道/顶管/钢板桩/降水/检查井/路缘石"等 → 市政
清单内有"模板/钢筋/砌体/抹灰/防水/脚手架" → 土建
清单内有"电气/暖通/消防/给排水/弱电" → 机电安装
清单内有"地砖/墙纸/木饰面/吊顶/精装" → 装饰装修
清单内有"乔木/草坪/水景/园路/小品" → 园林景观
（其它专业对照见下表）

### 第 2 步：查综合单价

打开相应专业的 `*_综合单价速查表.md`，按 Ctrl+F 搜项目名前 3-5 字。
 vinden 多个匹配项时优先选：
- 项目所在地相同/相近的来源（如广东项目先选 `49、中建集团23年四季度-市政道路排水工程指导价`）
- 报价时间最新（2023年4季度优先于 2022 年）
- 单位完全一致的列

### 第 3 步：核对单位（关键）

清单单位 → 与表头"单位"列对照：
- 一致 → 直接套用
- 不一致 → 必须换算并明示
  - 常见换算见 `查询存档\05_UNIT_AUDIT.md`
  - 钢板桩 t ↔ m²：见 `市政\重点专题_管道基坑钢板桩.md` 章节"钢板桩"

### 第 4 步：补二类、三类费用（详见 `规费措施税金\规费_措施费_税金费率表.md`）

- 二类-单价措施：模板、脚手架、降水等已含在"综合单价分析表"中
- 二类-总价措施：安全文明 + 夜间 + 二次搬运 + 冬雨季 = 约 4%~7%（合计）
- 三类-规费：人工费 × 规费费率（约 25%）
- 税金：(一+二+其他+规费) × 9%

### 第 5 步：综合汇总

```
工程总造价 = Σ (分部分项工程量 × 综合单价)
              + 措施费 (单价措施 + 总价措施)
              + 其他项目费
              + 规费
              + 税金
```

详细组成算法见 `查询存档\06_COST_COMPOSITION.md`。

## 四、关于本库与原文件的关系

- **本库**：从原几万份文件中提炼出"实体项目-单位-单价"三元组，已剔除合计/小计/费率
- **原文件**：保留在原位（`H:\AI-model\我的资源\` 与 `H:\AI-model\20、合同、方案...` 等），是本库的来源
- **追溯路径**：每条记录的"来源"列已注明文件名；如需回溯"`49、中建集团23年四季度-市政道路排水工程指导价.xlsx`"，原文件位于 `H:\AI-model\我的资源\1、土建工程价格数据库【2025.03月更新】\49、中建集团23年四季度-市政道路排水工程指导价.xlsx`
- **更新原则**：后续如新增月度信息价、季度指导价等，应同步：
  1. 在原位置放入新文件（保持原目录约定）
  2. 重新运行 `查询存档\_extract_prices_v2.py` 刷新专业速查表
  3. 重新运行 `查询存档\filter_municipal.py` 刷新专题表
  4. 重新运行 `查询存档\extract_regions.py` 刷新省市主线表

## 六、几个常用"快捷查法"

### 6.1 顶管单价（DN1200）

```
→ 价格信息库\市政\重点专题_管道基坑钢板桩.md
→ Ctrl+F：DN1200 / 顶管 / d1200
→ 命中的多个单价按"近区/近期/同口径"取用
```

### 6.2 钢板桩支护 (拉森 SP-Ⅲ, 桩长 9 m, 工期 90 天)

```
→ 价格信息库\市政\重点专题_管道基坑钢板桩.md → "钢板桩" 章节
→ 或 → 查询存档\07_PIPELINE_EXCAVATION_PRICE.md 第三节 → 含 t↔m² 换算实例
```

### 6.3 某广东项目策划阶段

```
→ 主要综合单价：价格信息库\<专业> → 优先选带"49、"标号的（中建集团23年4季度-市政道路排水工程指导价）
→ 信息价：价格信息库\信息价_分省市\省市信息价主线表.md → 25 广东 → 找地市
→ 规费：规费措施税金\规费_措施费_税金费率表.md → 广东省 0.2705
→ 安全文明：同上 → 广东省 3.16%
→ 税金：9%
```

### 6.4 给定工程量表，要算估算总造价

```
1. 每条清单行：到 <专业>速查表找单价 → 单位核对 → 工程量×综合单价=合价
2. 累加 = 分部分项合计
3. 二类-单价措施：在清单中"模板/脚手架/降水/钢板桩"等已含
4. 二类-总价措施：分部分项 × (安全文明 + 其它 1% ~ 1.5%) ≈ 分部分项 × 4%~7%
5. 三类规费：人工费 × 25% （或按当地费率）
6. 税金：(1+2+3+4+5+其它) × 9%
7. 总价 = 1+2+3+4+5+6
```
